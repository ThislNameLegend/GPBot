# Core Service (Go)

Основной модуль бизнес-логики приложения.

## Функции:
1. Управление опросами и вопросами
2. Хранение ответов пользователей
3. Предоставление API для веб-клиента и Telegram бота

## API Endpoints:
- `GET /api/health` - проверка работоспособности
- `GET /api/surveys` - список всех опросов
- `GET /api/surveys/{id}` - детали опроса с вопросами
- `POST /api/surveys/{id}/submit` - отправка ответов

## Запуск:

go run main.go
