# Веб-клиент (React)

## Функциональность:
1. Главная страница с описанием сервиса
2. Страница входа (моковая авторизация)
3. Список доступных опросов
4. Страница прохождения опроса

## Технологии:
- React 18
- React Router для навигации
- Axios для HTTP запросов
- CSS модули для стилей

## Запуск:
```bash
npm install
npm start
